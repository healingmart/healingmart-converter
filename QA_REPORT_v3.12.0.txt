4. Important limitations

- TAR/ZIP 변환은 안전을 위해 최대 500개 항목, 압축 해제 결과 약 150MB 제한
- TAR 특수 항목(심볼릭 링크·장치 등)은 ZIP 변환 시 제외될 수 있음
- GZ → ZIP은 GZ 단일 스트림을 풀어 하나의 파일로 ZIP에 저장
- 브라우저에서 실제 JSZip/Pako CDN 로딩 및 모바일 대용량 변환은 GitHub 배포 후 최종 확인 필요
