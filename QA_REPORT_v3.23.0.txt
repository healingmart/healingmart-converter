[ZIP → 7Z]
- ZIP을 가상 파일시스템에서 해제
- 항목/확장 크기 검사
- 7-Zip -t7z 방식으로 다시 압축

[7Z / RAR → ZIP]
- 7-Zip WebAssembly로 압축 해제
- 일반 파일/폴더를 표준 ZIP으로 재압축
- 암호화·손상 파일은 제한될 수 있음

[BZ2 / XZ → ZIP]
- 단일 압축 스트림을 해제
- 결과 파일을 ZIP 컨테이너 안에 저장

[ZIP → GZ]
- ZIP 내부 항목을 GZ로 변환하는 것이 아님
- ZIP 파일 바이트 자체를 GZIP으로 압축
- 출력명: 원본이 sample.zip이면 sample.zip.gz
- GZIP이 단일 데이터 스트림 형식이라는 특성에 맞춘 구현
