[주의]
ZIP → GZ는 ZIP 내부 파일을 하나의 GZ로 합치는 기능이 아니라
ZIP 파일 자체를 GZIP 압축하여 .zip.gz로 만드는 기능입니다.

7Z/RAR/BZ2/XZ 변환은 첫 실행 시 7z-wasm 1.2.0의
약 1.65MB WebAssembly 모듈을 추가로 불러옵니다.
