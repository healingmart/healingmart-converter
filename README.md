# HealingMart Converter Studio v3.0.0

계산기와 다른 "툴 전문 스튜디오"형 변환기 허브입니다.

## 홈 구조
1. 대표 변환 3개 쇼케이스
2. 파일 형식 자동 추천 드롭존
3. 인기 변환 6개 기본 노출 + 더 보기
4. 15개 카테고리 아코디언(한 번에 하나만 펼침)
5. 각 카테고리에서는 대표 8개만 미리보기 후 전체 페이지 이동

## 카테고리 페이지
- 인기 도구 우선
- 전체 목록은 처음 24개만 표시
- `더 보기`로 24개씩 추가
- 검색 지원

## 운영
- Blogger가 실제 사용자 화면
- GitHub Pages는 정적 CSS/JS/Registry 호스팅
- 엔진이 없는 서버형 변환기는 `준비 중` 표시
