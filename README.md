# HealingMart Converter Studio v3.5.0

계산기와 다른 "툴 전문 스튜디오"형 변환기 허브입니다.

## v3.5.0 핵심 변경
- 상단에 `원하는 변환 도구를 직접 찾아보세요` 접이식 빠른찾기 추가
- 빠른찾기 내부에 전체 카테고리, 자주 찾는 변환 6개, 즉시 검색 제공
- 빠른찾기는 기본 접힘 상태이며 필요할 때만 펼침
- 기존 대형 파일 드롭존은 그대로 유지
- 하단 인기 변환과 카테고리 탐색 구조도 그대로 유지
- 하단 카테고리 아코디언은 처음에는 모두 접힌 상태로 변경
- Blogger PC 화면 우측 쏠림 방지를 위해 강제 left/transform 중앙정렬 제거 및 부모 폭 초기화

## 사용자 동선
1. 형식을 알고 있음 → 상단 빠른찾기
2. 무엇으로 변환할지 모름 → 큰 파일 드롭존
3. 둘러보고 싶음 → 하단 인기 변환 / 카테고리

## 운영
- Blogger가 실제 사용자 화면
- GitHub Pages는 정적 CSS/JS/Registry 호스팅
- 서버 엔진이 없는 변환기는 `준비 중` 표시
