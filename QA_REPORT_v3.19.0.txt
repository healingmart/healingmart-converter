[출력 방식]
BMP
- 24비트 BI_RGB 비압축 BMP
- 행 4바이트 정렬
- 투명 PNG 입력은 선택 배경색으로 합성

ICO
- 최대 256×256
- 1개 아이콘 엔트리 생성
- 내부 이미지 데이터는 PNG
- PNG 투명 배경 유지 가능

GIF
- gifenc 1.0.3 사용
- 단일 정지 프레임 GIF 생성
- 최대 256색
- PNG 투명도는 GIF 1비트 투명도로 단순화 가능
- GIF 변환은 1,600만 픽셀 이하 제한

[준비 중 유지]

- PNG → AVIF
